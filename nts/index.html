Troubleshooting

If you can't install, check your Internet connection, and try again.
If update site is unavailable and it is hosted on GitHub Pages, for example Nodeclipse, check GitHub System Status.
Check home page for announcements.
Try support options.
If you notice that something is buggy or slow (e.g. GFM View) _just don't use it_. If you really care, help us to improve it.
