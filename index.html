@nodejs

Nodeclipse is an plugin that adds IDE functionality to the Eclipse for the Node.
The purpose of Nodeclipse is to create an environment in which development of Node is easy for a beginner to intermediate user. Development is slow, but ongoing. If you would like to help out, please leave a comment stating so in the Nodeclipse forum.
